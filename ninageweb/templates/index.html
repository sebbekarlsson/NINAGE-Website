NINAGE

The Free Interactive Media Engine

An engine, not just for Games

Ninage is not a Game Engine.

Ninage can be used to create games and other interactive media in 2D or 3D using C++.

History

The Ninage project started Wed Jan 4 20:23:39 2017 with the filosofy of being free, free as in free speech.

Unlike other engines such as Unity or Game Maker, Ninage is completely open source and does not contain any malicious code.

Contribution
